# Internal CRM Integration Guide (sample excerpt)

## Раздел 4.2 — Битрикс24
Внутренняя CRM поддерживает официальный коннектор Битрикс24 (версия 2.1). Синхронизация
контактов и сделок выполняется в реальном времени через REST API. Поддерживаются
как облачные, так и on-premise установки Битрикс24.

## Раздел 3.1 — Анализ тональности
Анализ тональности доступен только в постобработке после завершения звонка.
Real-time анализ эмоций абонента в текущей версии не поддерживается.

## Раздел 5 — Поддерживаемые протоколы интеграции
Поддерживаются протоколы интеграции: REST API, SOAP, Webhooks, SFTP.
Поддержка SAP RFC напрямую не предусмотрена и требует middleware.
