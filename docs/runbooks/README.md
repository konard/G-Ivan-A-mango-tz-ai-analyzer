# 📚 Runbooks (Эксплуатационные инструкции)

## Назначение
Каталог для инструкций по эксплуатации и поддержке системы на этапе **Пилот** и **Production**.

## Планируемые документы (Пилот, недели 3–5)
- `incident-response.md` — действия при падении точности < 70%.
- `kb-update.md` — процесс обновления базы знаний.
- `llm-failure.md` — что делать при недоступности всех LLM-провайдеров.
- `ba-validation-guide.md` — гайд для бизнес-аналитиков по проверке результатов ИИ.

## Статус
На этапе **MVP** каталог пуст. Наполнение начнётся после успешного завершения MVP и перехода на этап «Пилот» (см. [`docs/CONCEPT.md`](../CONCEPT.md), раздел 7, и [`docs/standards/roles.md`](../standards/roles.md), раздел 4).
