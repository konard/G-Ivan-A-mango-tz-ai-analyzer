# 📘 Справка для БА — Руководство пользователя

Это руководство для **Бизнес-Аналитиков (БА)**, которые работают с системой
**Clarify Engine** — инструментом для анализа тендерных ТЗ и консультаций по
базе знаний.

Руководство **не описывает архитектуру и код**. Только действия пользователя
в интерфейсе и ожидаемый результат.

## Глоссарий: 5 терминов, которых хватит на 90% работы

| Термин | Что это |
|--------|---------|
| **ТЗ** | Техническое задание — документ с требованиями от заказчика (`.xlsx` или `.docx`) |
| **Режим работы** | «📊 Анализ ТЗ» (массовая проверка файла) или «💬 Консультация» (диалог) |
| **Статус** | Вердикт системы по требованию: `Да` / `Нет` / `Частично` / `НД` / `Ошибка` |
| **Confidence** | Уверенность модели в вердикте: число от 0 до 1. Порог ≥ 0.85 — автопринятие |
| **RunID** | Уникальный идентификатор запуска. Сообщайте его в поддержку при проблемах |
